Virtual environment for flask:      
$ python3 -m venv venv      

Activate the environment:       
$ . venv/bin/activate       
> venv\Scripts\activate     

Run Locally:        
> $env:FLASK_APP = "flask"
> $env:FLASK_ENV = "development"
> flask run
