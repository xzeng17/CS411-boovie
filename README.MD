Virtual environment for flask:      
$ python3 -m venv venv      

Activate the environment:       
$ . venv/bin/activate       
> venv\Scripts\activate     

Run Locally:        
$ export FLASK_APP=flaskr && export FLASK_ENV=development && flask run      
> $env:FLASK_APP = "flaskr";$env:FLASK_ENV = "development";flask run
